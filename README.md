# Tecnologias para Desenvolvimento de Soluções de Big Data

Este repositório reúne códigos, microatividades, dados e documentação
para o curso/projeto de **Tecnologias para Desenvolvimento de Soluções
de Big Data**. O objetivo é centralizar recursos práticos e teóricos
para aplicação de técnicas de Big Data.

## Conteúdo do Repositório

-   `dados/` --- diretório que contém bases de dados utilizadas nas
    atividades.
-   `documentacao/` --- anotações, textos explicativos, slides ou
    artigos de suporte.
-   `microatividades/` --- scripts menores para praticar conceitos de
    Big Data durante o curso.
-   `trabalho_pratico/` --- projeto prático mais complexo envolvendo o
    uso das tecnologias estudadas.
-   `executar_todas_microatividades.py` --- script Python para executar
    automaticamente todas as microatividades.
-   `requirements.txt` --- dependências Python necessárias para rodar os
    scripts deste repositório.

## Tecnologias Utilizadas

-   **Python** --- para scripts e automação.
-   **Jupyter Notebooks** --- para análises exploratórias e relatórios
    interativos.

## Como Começar

1.  Clone este repositório:

    ``` bash
    git clone https://github.com/VictorPFe/Tecnologias-Para-Desenv.-de-Solucoes-de-Big-Data.git
    ```

2.  Entre na pasta do projeto:

    ``` bash
    cd Tecnologias-Para-Desenv.-de-Solucoes-de-Big-Data
    ```

3.  Crie um ambiente virtual:

    ``` bash
    python -m venv venv
    source venv/bin/activate   # Linux/macOS
    venv\Scripts\activate    # Windows
    ```

4.  Instale as dependências:

    ``` bash
    pip install -r requirements.txt
    ```

5.  Execute as microatividades:

    ``` bash
    python executar_todas_microatividades.py
    ```
